## IMPOSSIBLE? None.

All ideas are technically feasible. Some are just more complex:
- **Plaid** = paid API ($300-500/mo at scale) but doable
- **PC app** = significant effort but not impossible
- **Location reminders** = battery concerns, but standard feature
