# Task Queue

## How This Works
- Tasks have status: `OPEN` → `CLAIMED:W#` → `DONE:W#`
- Before starting work, change status to `CLAIMED:W#` (your worker number)
- When finished, change status to `DONE:W#`
- Only claim tasks in YOUR domain
